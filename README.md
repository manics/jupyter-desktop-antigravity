# Jupyter Desktop Antigravity

[![Build](https://github.com/manics/jupyter-desktop-antigravity/actions/workflows/build.yaml/badge.svg)](https://github.com/manics/jupyter-desktop-antigravity/actions/workflows/build.yaml)

A container image providing the [Google Antigravity IDE](https://antigravity.google/) running in Jupyter, built on top of [Jupyter Desktop MATE](https://github.com/manics/jupyter-desktop-mate/).

## Usage

Edit and run [`./antigravity-container`](./antigravity-container),
then open `localhost:5901` in a VNC client.

Alternatively see https://github.com/manics/jupyter-desktop-mate/ but use
`ghcr.io/manics/jupyter-desktop-antigravity:latest` as the image.
